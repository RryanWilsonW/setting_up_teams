  ## Description
  Project Description: Setting up teams is a command line application that can be used to generate a rouster of the employees in your team.
  
  Installation Instructions: After cloning the directory, you will have to do an NPM install to get the packages required to create the html file.
  
  Usage Information: To use this program you will call a node index.js, then you will be prompted with some questions to get your teams information. When you are all finished, hit finish application and the new html file will be generated.
